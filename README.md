# Salary Prediction Using Polynomial Regression

## Project Overview
This project predicts employee salary based on years of experience using Polynomial Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Workflow
1. Load Dataset
2. Data Preprocessing
3. Train Polynomial Regression Model
4. Predict Salary
5. Visualize Results

## Model Used
Polynomial Regression

## Dataset
Years of Experience vs Salary
